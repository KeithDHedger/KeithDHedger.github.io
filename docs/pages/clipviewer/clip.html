Clipboard Viewer

Download this simple command line app to show what's on the global clipboard. Compile with the usual linux ./configure;make;sudo make install.
It can be run from the command line or double clicked and it will show a small window with the clipboard contents, either a picture. file name or text. Command line usage: clipviewer --nogui print the text on the clipboard to stdout instead of opening a window, for use in scripts (nothing will be printed if the clipboard holds a picture), but if it does a png image file will be created in the current folder.
