GUI's & Apps
These programs are (obviously!) GUI wrappers around various scripts/programs and small programs they will have to be compiled from source, you will need automake1.9 etc. to compile the programs.
Where autogen.sh is used you can add a prefix for the installation i.e. ./autogen.sh --prefix=/somefolder/someotherfolder.
The default is to use --prefix=/usr/local
