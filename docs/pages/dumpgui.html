Dump GUI

NO LONGER SUPPORTED!
This is a GUI for the dump/restore program for linux, a slightly altered version of the restore program is used for extracting individual files so you must compile and install Dump/Restore first, you should also remove any previously installed version. you will also need to have dvd+rw-tools (they should be in your distros repos) installed.
Dump GUI is a work in progress and may (will?) have some bugs, also some features like dumping and restoring to smb are experimental.
To build:
./configure --prefix=/usr/local
make
sudo make install
Download Dump/Restore

Then make the GUI:
./autogen.sh --prefix=/usr/local
make
sudo make install
Download DumpGUI
